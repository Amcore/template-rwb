# react-webpack-boilerplate

> A SPA boilerplate with React, built with love.

## Features
 - **React** 15.4.2
 - **JSX**
 - **ES6**
 - **Webpack**
 - **Express** the dev-server.
 - **Hot-Reload**
 - **Proxy**
 - **Global-Variable**
 - **eslint**

## Usage

```bash
# install sao first
yarn global add sao

# install all this dependencies.
npm install

# development, default port: 8080
npm run dev

# production
npm run build
```
